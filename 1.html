여기는 head~~

  • <!DOCTYPE html> 혹은 <!doctype html>
  • <html> 태그로 전체 감싸기
  • 한글 깨질 땐 <meta charset = “utf-8”>
    : 홈페이지야 이 웹페이지는 UTF-8로 만들어졌기 때문에 UTF-8로 열어야 해
  • <title>홈페이지 이름</title>

여기는 body~~

  • <h1></h1>

    <h2></h2>

    <h3></h3>

    <h4></h4>

    <h5></h5>
    <h6></h6>
  • 강조: <strong></strong>, <b></b>
  • underline: <u></u>
  • 태그는 중첩 사용 가능
  • 줄바꿈: <br>
  • 단락: <p> -> 단락과 단락의 간격 고정되어 있음. (시각적으로 자유도 떨어짐)
       +) css 문법: <p style="margin-top:45px;"> 태그 위쪽에 45px만큼 여백(margin) 생김
  • 이미지: <img>
    <img src="이미지주소.png">
    인터넷에 있는 이미지 써도 됨. 굳이 내 컴퓨터에 있는 이미지 안 써도 됨

    내 컴퓨터에 있는 이미지 쓰는 방법

    • 1. 다운 받은 이미지를 프로젝트 폴더에 위치
    • 2. 같은 위치에 있는 이미지 파일이라면 src 속성값으로 이미지 파일의 이름만
      예: <img src ="coding.jpg">

    이미지 크기 설정
    • width의 값으로 숫자나 % 사용하면 됨.
      예: <img src="이미지 주소" width="50%">

    • a: anchor, href: HyperText Reference

웹사이토~

  • 웹사이트(web site): 링크를 통해서 서로 결합되어 있는 웹페이지들의 그룹
  • 웹 = 인터넷의 부분집합
  • 웹호스팅 -> netlify.com
    1. 새로운 사이트 만들려면 Sites 메뉴 클릭
    2. 맨 밑에 점선 박스 or Add new site에서 Deploy manually(업데이트할때도)
    3. 업로드 할 파일엔 올릴 html 파일이 꼭 있어야 함
    4. 폴더 Drag and drop
    5. 주의!!! 안될때는 index.html 파일이 있는지 확인하기 (파일명 그대로)
  • 웹서버 -> 생활코딩 강의 참조^^
