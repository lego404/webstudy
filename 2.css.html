    • <style>

      //a 태그의 폰트의 색을 blue로 하겠다
      a{    <!--선택자-->
          color: blue;    <!--효과-->
      }

      </style>


    • h1{

style 태그 외에 CSS 표시하기

  • 속성 이용하기
    • 웹 페이지에서 여러 개의 a 태그 중 하나만 다른 색깔을 바꾸는 건 style을 쓰기 애매함
    • 그래서 <a href = "css.html" color: red>CSS</a> 사용
    • 하지만 이렇게 쓰면 웹브라우저는 이게 html인지 css인지 모름
    • 그래서 <a href = "css.html" style = "color: red">CSS</a>
    • style 태그를 직접 사용하는 경우, 선택자가 필요없음

++추가

  • 주소에서 밑줄 제거하기
    • a 태그에서 text-decoration: none; //장식이 없다
  • 특정 태그만 밑줄 유지하기
    • 태그 안에서 style = "color: red; text-decoration: underline" 추가해주기. (세미콜론으로 구분)

특정 글자 크기 키우기

  • hi 선택자 안에 font-size: 60px(혹은 %)

가운데 정렬

  • text-align: center

응용

  • 목표
    1. 모든 링크는 기본적으로 검은색
    2. 사용자가 방문했던 링크는 회색
    3. 현재 방문중인 링크는 빨간색
  • 방법
    1. a 선택자:
      color: black;
    2. 태그 직접 속성:
      다른 애들은 전부 태그안에서 style = "color:gray" -> 하지만 이는 코드중복
      따라서 A, B, C 중 A, B를 그룹으로 묶어서 폰트 컬러 바꿀거임
      예)
      < a href = "1.html" class = "saw">HTML
      < a href = "2.html" class = "saw">CSS
      --------------------------------------------------
      선택자 saw 만들기
      .saw {color: gray;}
      //. 안 붙이면 그냥 이름이 saw인 태그들 바꾸는건데
      //. 붙이면 class 이름이 saw인 태그들 바꿈

      • 클래스 중에 하나만 색 바꾸기

        • 예)
          < a href = "1.html" class = "saw">HTML
          < a href = "2.html" class = "saw active">CSS
        • .active { color: red; }

      • 우선순위

        • saw 클래스보다 active 클래스가 더 나중에 선언되었음.
          -> 더 나중에 선언한 명령이 영향력을 가짐.
        • id 선언
          • 예)
            < a href = "2.html" class = "saw" id = "active">CSS
          • #active {color: red;}
        • 우선순위: id 선택자(active) > 클래스 선택자(saw) > 태그 선택자(a)
        • 다 같은 태그일 때 우선순위: 제일 마지막에 선언될수록 우선순위 높음
      • ++추가
        • id는 한 번만 언급해야 한다.

                    /*

                        display:inline;
                    }
                    /*
                    자기 자신의 크기만큼을 갖는 태그들
                    inline element
                    */
                    a {
                        border-width: 5px;
                        border-color: red;
                        border-style: solid;
                        
                        /*반대로 블록 레벨 태그로 바꿔주기*/
                        display:block;
                    }
                </style>
